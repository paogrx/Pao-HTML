Atributos para tipo Number

Para usar un campo numerico con rangos, utilizamos los atributos MIN y MAX,para determinar el valor inicial y el limite máximo

El atributo STEP, especifica el incremento o disminución que tendrá


El atributo MAXLENGTH, limita la cantidad de caracteres que se pueden ingresar y el atributo MINLENGTH es la cantidad minima

El atributo pattern se utiliza para especificar una expresion regular que debe cumplirse para que el campo de entrada sea valido, por ejemplo:.

Atributo PLACEHOLDER y REQUIRED

El PLACEHOLDER sirve para agregar sugerencias en el llenado del formulario, como un apoyo visual, se imprime dentro de los campos y al hacer clic desaparece (solo es visible si esta vacío) por su lado REQUIRED sirve para realizar una vadilacion donde no podemos enviar los datos si esta vacio (solo funciona usando el elemento form y un botom que funciones como detonante).
