Estructura Básica

HTML

Indica el inicio de un documento HTML y contiene las demás etiquetas, es decir , es la raíz del documento

HEAD

Contiene metadatos del documento HTML, como el titulo, codificación de caracteres, estilos, scrips, etc .Este contenido no se muestra en la pagina, pero es importante para la presentación y el funcionamiento del documento


BODY

es donde está el contenido principal de HTML, como texto, imagenes, enlaces, videos, audios, formularios, entre otras ,es decir , lo que se muestra en pagina web
