Etiquetas básicas de HTML

Las etiquetas de HTML sirven para definir la estructura y el contenido de una pagina web.A continuación se presentarán algunas para dar formato e importancia al texto:

Etiqueta H1 al H6

es considerada como una etiqueta para dar formato a los titulos o temas de nuestro contenido, donde el H1 se usa para dar más improtancia con un tamaño y un grosor mayor al texto del contenido, es decir el titulo principal, hasta el H6 usada para titulos o secciones de los temas principales.ejemplos:

Etiqueta B,I,U y S

Conocidas como estilos de texto:

B: Es conocida como negritas, da enfasis a un texto, cambiando su grosor

I: Es conocida como Cursiva, dá enfasis a un texto, cambiando la inclinación

U: Es conocida como subrayado, esta etiqueta agrega una linea debajo del texto.

S: Es conocida como Tachado, agrega una linea en la parte media del texto.

Etiquetas STRONG, EM y MARK

Conocidas como etiquetas de énfasis e importancia.

STRONG: Le da mayor relevancia o importancia al texto, es decir es una etiqueta semántica, Aumenta el grosor de la letra.

EM: Le da énfasis a la palabra, haciendo que el texto se incline y tambien se considera una etiqueta semántica

MARK: Conocida como Marcado, sirve para resaltar el texto.
