CSS

Como HTML, CSS (Cascading style sheets) usa hojas de estilo en cascada en español, no es realmente un lenguaje de programación, tampoco es un lenguaje de marcado. Es un lenguaje de hojas de estilo, es decir, te permite aplicar estilos de manera selectiva a elementos en documentos HTML. se usará la versión CSS3.

Para enlazar un archivo HTML y CSS, se agrega una linea donde se indica el tipo de archivo y la ubicación: link href= "folder/archivo_Estilo.css" rel= "stylesheet" type="text/css"
