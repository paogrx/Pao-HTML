Funciones Comunes en CSS

1. Funciones de Color

  • rgb() - Define colores con valores de rojo, verde y azul.
  • rgba() - Define colores con valores de rojo, verde, azul y transparencia.
  • hsl() - Define colores con matiz, saturación y luminosidad.
  • hsla() - Igual que hsl() pero con transparencia.
  • hex() - Define colores con un valor hexadecimal (por ejemplo, #ff0000 para rojo).

2. Funciones de Transformación

  • rotate() - Rota un elemento en un ángulo específico.
  • scale() - Escala un elemento en los ejes X y/o Y.
  • translate() - Desplaza un elemento a lo largo de los ejes X y/o Y.
  • skew() - Inclina un elemento en los ejes X y/o Y.
  • matrix() - Aplica una transformación 2D con una matriz de transformación.

3. Funciones de Gradientes

  • linear-gradient() - Crea un degradado lineal entre dos o más colores.
  • radial-gradient() - Crea un degradado radial que se expande desde el centro.
  • conic-gradient() - Crea un degradado cónico alrededor de un punto.

4. Funciones de Medición

  • calc() - Realiza cálculos matemáticos para establecer valores de CSS.
  • min() - Selecciona el valor más pequeño de entre dos o más expresiones.
  • max() - Selecciona el valor más grande de entre dos o más expresiones.
  • clamp() - Restringe un valor dentro de un rango mínimo y máximo.

5. Funciones de Forma y Clip

  • circle() - Define una forma circular.
  • ellipse() - Define una forma elíptica.
  • inset() - Define un área rectangular.
  • polygon() - Define un área de recorte usando un polígono personalizado.

6. Funciones de Fuente

  • var() - Utiliza variables de CSS.
  • attr() - Utiliza el valor de un atributo HTML como contenido CSS.
